## instalação de dependências


Instale as dependências

```
sudo apt-get install ruby1.9.3 nodejs
```


Instale o jekyll e algumas outras dependências

```
sudo gem install jekyll jekyll-tagging
```



## Rodando

Para rodar o site 

```
jekyll build
jekyll serve
```

acesse `http://localhost:4000/`
